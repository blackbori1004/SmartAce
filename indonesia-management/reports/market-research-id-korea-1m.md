# Market Research (1M): Indonesian 20s Travelers to Korea

Date: 2026-03-01
Scope: Recent 1 month (Google Trends proxy + official context from KTO/MOFA)

- KTO Jakarta note: Indonesian arrivals to Korea reached **365,596 in 2025** (record high).

## 3) What this means for your target (Indonesian 20s)

Likely real behavior flow:
1) **Admin/planning first** (visa, KVAC, cost checks)
2) **Place-locking next** (Myeongdong, Hongdae)
3) **Experience curation** (night market, N Seoul Tower, café/street route)
4) **Booking timing** near departure (flight/hotel checks)

Experience preference pattern:
- High social-share value (street, night market, skyline)
- Convenient cluster routing (Myeongdong + Namsan/N Seoul Tower)
- Youth district add-on (Hongdae)
- Optional day-trip for second/third day (Nami)

## 4) Product design implication (One-day package)

Recommended best-fit one-day cores:

### Core Route A (Most practical)
Myeongdong -> N Seoul Tower -> Hongdae night
- Why: aligns with strongest place-intent + night experience demand.

### Core Route B (Culture + photo)
Gyeongbok/Bukchon -> Ikseondong -> Myeongdong night market
- Why: balances "Korea classic" + trendy close-out.

### Core Route C (Nature-lite add-on)
Seoul Forest/Hangang -> Seongsu café -> Hongdae
- Why: 20s aesthetic + lower fatigue.

## 6) Cautions

- Google Trends values are **index-based** (relative), not absolute search volume.
- Some keywords (e.g., hongdae) include non-travel noise; interpret by pattern, not single-day spikes.
- For pricing decisions, combine with OTA click/cost data once ads go live.
