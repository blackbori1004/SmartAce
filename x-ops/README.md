# X 운영 플랜 (Aegis Ace)

## 목표
- 1단계: 읽기 전용 리서치 + 초안 생성 자동화
- 2단계: 반자동 게시(승인 후 게시)

## 운영 원칙
- 승인 없는 자동 게시 금지
- 금지 주제/민감 키워드 회피
- 과도한 빈도 금지

## 1단계 (즉시)
- 6시간 주기 초안 생성
- 톤: 신뢰형, 간결, 240자 이내
- 출력: 본 채팅으로만 전달

## 2단계 (API 연결 후)
- 승인 키워드 수신 시에만 게시
- 게시 전 최종 검수 체크:
  1) 사실관계
  2) 규정 위반 표현
  3) 과장/투자 조언성 문구

## 필요한 비밀값
- X API Bearer/Access 토큰류
- 절대 채팅 본문에 평문 공유 금지
- `.env.x` 파일로만 저장
